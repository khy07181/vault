### 음악 추천 & 탐색

- Discover Weekly:
  - 매주 취향 기반 추천 플레이리스트 제공
- Release Radar:
  - 팔로우한 아티스트 중심 신곡 추천
- 라디오 기능:
  - 특정 곡/아티스트 기준 자동 플레이리스트 생성
  - 새로운 음악 찾기에 가장 효율적
- 추천 루틴:
  - 좋아하는 곡 → 라디오 → 마음에 드는 곡 저장 → 반복

---

### 신곡 확인 방법

- 아티스트 팔로우:
  - 좋아하는 아티스트를 팔로우하면 업데이트 추적 가능
- 알림(종 아이콘):
  - 팔로우한 아티스트의 신곡만 모아서 확인 가능
  - Release Radar보다 정확한 신곡 관리 가능
- 주간 루틴:
  - 매주 금요일 확인 추천

---

### 플레이리스트 활용

- 플레이리스트 + 앨범 조합:
  - 플레이리스트는 셔플
  - 앨범은 셔플되지 않아 곡 흐름 유지 가능
- 플레이리스트 폴더 (PC):
  - 여러 플레이리스트를 폴더로 정리 가능
- 모바일 플레이리스트 고정:
  - 자주 듣는 플레이리스트 상단 고정 가능

---

### 숨겨진 UI / 제스처

- 스와이프 좋아요:
  - 곡에서 오른쪽 스와이프하면 바로 좋아요
- 작은 플레이어 제스처:
  - 스와이프로 다음/이전 곡 이동 가능
- 검색창 더블탭:
  - 검색 아이콘 두 번 클릭 시 바로 입력

---

### 음질 & 설정

- 오디오 노멀라이제이션 OFF:
  - 전체 볼륨이 더 커짐
  - 대신 곡 간 음량 차이는 존재

---

### 숨겨진 콘텐츠 탐색

- 과거 Wrapped 보기:
  - 검색창에 spotify:genre:201# 입력
- Decades 기능:
  - 80s, 90s, 2000s 등 시대별 음악 탐색 가능

---

### 기타 사용 팁

- 홈 화면 하단:
  - 팟캐스트 아래 음악 추천 영역 존재
- last.fm 연동:
  - 청취 기록 및 취향 분석 가능
