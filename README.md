# 개인 지식 저장소 (My Personal Vault)

이곳은 [Obsidian](https://obsidian.md)을 사용하여 관리하는 개인적인 지식 저장소입니다. 

제 생각, 학습 내용, 그리고 흥미로운 정보들을 기록하고 연결하여 저만의 "두 번째 뇌(Second Brain)"를 구축하는 공간입니다.

## 🗂️ 폴더 구조

이 저장소는 PARA(Projects, Areas, Resources, Archives) 메소드를 기반으로 다음과 같이 구성되어 있습니다.

- **`1_Project/`**: 명확한 목표와 기한이 있는 진행 중인 프로젝트 관련 파일들을 보관합니다.
- **`2_Area/`**: 건강, 취미 등 지속적으로 관리해야 하는 삶의 각 영역에 대한 파일들을 저장합니다.
- **`3_Resource/`**: 관심 있는 특정 주제(예: 프로그래밍, 디자인, 심리학)에 대한 자료들을 모아둡니다.
- **`4_Archive/`**: 완료된 프로젝트, 더 이상 관리하지 않는 영역, 오래된 리소스 등 현재는 활발하게 사용하지 않는 파일들을 보관합니다.
- **`5_Fleeting/`**: 순간적으로 떠오른 아이디어나 정제되지 않은 생각들을 빠르게 기록하는 공간입니다.

## ✨ 목적

이 Vault의 주된 목적은 다음과 같습니다.

- **기록**: 일상적인 생각, 아이디어, 학습한 것들을 꾸준히 기록합니다.
- **연결**: 기록된 정보들을 서로 연결하여 새로운 통찰과 아이디어를 발견합니다.
- **성장**: 지식을 체계적으로 축적하고 발전시켜 개인적인 성장의 자산으로 삼습니다.
